What is the Document Object Model (DOM)?

The Document Object Model (DOM) is the data representation of the objects that comprise the structure and content of a document on the web.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future.

How do you manipulate an element's attributes and styles using the DOM?

You can dynamically manipulate elements on a webpage by interacting with their attributes and styles using the Document Object Model (DOM). This allows you to control the behavior and appearance of elements, enhancing user interaction and design flexibility.
